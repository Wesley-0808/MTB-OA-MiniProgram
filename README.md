# 🎉 About

- 一个微信小程序端的『企业级』管理系统（定制系统，非模板）
- 顺德中专团委学生会媒体部定制、授权使用
- 使用『[TDesign](https://github.com/Tencent/Tdesign-miniProgram)』组件库

# 🔨 Develop

```shell
# Install
npm install

# 构建npm
微信开发者工具-工具-构建npm
```
